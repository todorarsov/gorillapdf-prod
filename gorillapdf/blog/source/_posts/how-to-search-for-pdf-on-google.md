Welcome back to another PDF tutorial. We will show you how to search for PDF files in search engines like Google.

We already presented the popularity of the Portable Document Format in our infographic about [10 PDF Facts](https://gorillapdf.com/blog/10-facts-about-pdf-infographic/ "Ten Facts About PDF File Format").

So there may be many reason why you need to filter PDF results only. Obviosly, you need the information in a PDF format, and we'll cover that part here. 

More to that, you will learn how to search for other files types as well such as DOCX, TXT etc.

Let’s see the two methods for searching PDFs on Google. 
<!-- more -->
<!-- toc -->
<h2>Method 1 - Using Search Operator</h2>

Search Engines like Google allows you to search file types, and get the information faster by using **search operators**.

<h3>filetype:pdf</h3>

**Example 1:** filetype:pdf "spacex next launch"

It means to filter out PDFs that contain the phrase spacex next launch.

**Example 2:** filetype:pdf programming -"java"

It means to filter out PDF files about programming that don’t contain the word java.

Try it out.
<h2>Method 2 - Using Google Advanced Search</h2>

This method is basically a setup of the first method without typing. 

We will use the advanced search function in Google.

**Step 1)** Type your query in Google and click on Advanced Search.

{% asset_img img-fluid click-on-advanced-search-google.png "Click on Advanced Search in Google'" %}

**Step 2)** Scroll down to "Then narrow your results by..." and under "file type", **pick Adobe Acrobat PDF (.pdf)** option.

{% asset_img img-fluid filetype-pdf-advanced-search.png "Pick Adobe Acrobat PDF in file type'" %}

**Step 3)** Click on Advanced Search button and you will be directed to the search listings.

{% asset_img img-fluid search-results-filetype-pdf.png "Search listing with PDFs only'" %}

There you go. Google serves you only PDF documents with your desired query.

<h2>Closing Up</h2>
